This repository is to demo a potential product that enables greenhouse gas owners to operate electric boilers as an additional cost-effective source of heat.

TO VIEW DEPLOYED APP

The resulting demo can be viewed on electric-greenhouse-heating.streamlit.app. To make and view changes, please run the demo locally.

TO RUN LOCALLY

1. Ensure Python 3.7 or newer is installed on your system.

2. Install the required libraries: 
	
	pip install streamlit plotly numpy pandas
	
3. Clone this repository or copy the code files to your local machine.

4. Navigate to the folder Sympower-demo-master and run the app using the following command: 

	streamlit run run_streamlit.py
	
5 The app should aotumatically open in a new browser tab on default URL http://localhost:8501. If not, navigate there manually.

Any changes you make to the code will be visible in the resulting app.
